# Integrated NCD Care Tracker — DHIS2

A DHIS2 Tracker program for non-communicable disease patient 
follow-up at facility level, built for the Kenya health system context.

## What's in this repo
- `ncd-tracker-metadata.json` — Full DHIS2 metadata export

## Features
- Individual patient tracking across 4 program stages
- 8 clinical decision-support program rules
- Automated alerts for overdue patients, abnormal BP, missed refills
- Indicators for retention rate and BP control %

## Clinical Grounding
Thresholds based on Kenya MoH NCD Clinical Guidelines 2020.
Designed by a practicing Clinical Officer.
